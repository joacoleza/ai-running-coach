# Quick Plan: Remove dates from training plan, use day labels A-G and global sequential week numbers

## Objective

Refactor the entire training plan data model to remove calendar dates. Days use labels "A"-"G" within their week, weeks use globally sequential numbers (1, 2, 3...) across all phases. API routes, system prompt, XML tags, and frontend all updated accordingly.

4. **`web/src/hooks/usePlan.ts`** — Update `PlanDay` interface: remove `date`, add `label: string`. Update `PlanWeek` interface: remove `startDate`. Update `updateDay` to use `(weekNumber: number, label: string, ...)` signature calling `PATCH /api/plan/days/${weekNumber}/${label}`. Update `deleteDay` similarly. Update `addDay` to send `{weekNumber, label, ...}` in body to `POST /api/plan/days`.
