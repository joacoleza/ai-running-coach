   ```
   **This past-date allowance is ONLY for the initial `<training_plan>` block.** The `<plan:add>` command still cannot target past dates.
   ```
   with:
   ```
   **This past-date allowance applies to both the initial `<training_plan>` block AND the `<plan:add>` command** — as long as `completed: true` or `skipped: true` is set on the day.
   ```

   Also add an example to the `<plan:add>` tag table (~line 87) showing a past completed run:

   | `<plan:add date="2026-03-25" objective_kind="distance" objective_value="5" objective_unit="km" guidelines="Easy recovery run" completed="true" />` | Record a past completed run |
