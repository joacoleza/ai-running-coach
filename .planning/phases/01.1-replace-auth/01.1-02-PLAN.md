    **Create `web/src/pages/PasswordPage.tsx`** (per D-01, D-02):

    A form component with:
    - State: `password` (string), `error` (string | null), `loading` (boolean)
    - A single password input field (`type="password"`, `placeholder="Enter password"`)
    - A submit button ("Unlock" or "Enter")
    - On submit:
      1. Set `loading: true`, clear `error`
      2. Call `fetch('/api/health', { headers: { 'X-App-Password': password } })`
      3. If response is OK (200): call `localStorage.setItem('app_password', password)`, then call `onSuccess()` callback prop
      4. If response is 401: set `error` to `"Wrong password"`, clear password input
      5. If response is 503: set `error` to `"Service locked. Contact administrator."`
      6. Set `loading: false`
    - Props: `{ onSuccess: () => void }`
    - Styling: centered on page, Tailwind classes — `min-h-screen flex items-center justify-center bg-gray-50`, form in a white card with `bg-white rounded-lg shadow-md p-8 w-full max-w-sm`
    - Error displayed in red text below form: `text-red-600 text-sm mt-2`
    - Title: "AI Running Coach" in `text-xl font-bold text-gray-900 mb-6`

    **Update `web/src/App.tsx`** (per D-01, D-03):

    1. Add import: `import { PasswordPage } from './pages/PasswordPage'`
    2. Add state: `const [authenticated, setAuthenticated] = useState(() => !!localStorage.getItem('app_password'))`
    3. Add import: `import { useState, useEffect } from 'react'` (if not already imported)
    4. Before the `<BrowserRouter>` block, add a conditional:
       ```
       if (!authenticated) {
         return <PasswordPage onSuccess={() => setAuthenticated(true)} />
       }
       ```
    5. This means the entire router + AppShell is only rendered when authenticated is true

    For D-03 (401 handling), add a global fetch interceptor or handle in a wrapper. Simplest approach:
    - Create a `useEffect` in App that overrides `window.fetch` to intercept 401 responses:
      ```
      useEffect(() => {
        const originalFetch = window.fetch;
        window.fetch = async (...args) => {
          const response = await originalFetch(...args);
          if (response.status === 401) {
            localStorage.removeItem('app_password');
            setAuthenticated(false);
          }
          return response;
        };
        return () => { window.fetch = originalFetch; };
      }, []);
      ```
    - This ensures ANY 401 from ANY API call triggers redirect to password page (per D-03)
