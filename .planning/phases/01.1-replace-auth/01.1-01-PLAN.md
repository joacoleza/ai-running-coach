       - Signature: `export async function checkBlocked(): Promise<boolean>`
       - Reads from collection `auth`, document `{ _id: "lockout" }`
       - Returns `true` if `blocked === true`, `false` otherwise
       - If document doesn't exist, returns `false` (not blocked)

    3. **`requirePassword(req: HttpRequest)` function** (per D-05, D-06, D-09, D-10, D-11, D-12):
       - Signature: `export async function requirePassword(req: HttpRequest): Promise<HttpResponseInit | null>`
       - Returns `HttpResponseInit` (error response) if auth fails, or `null` if auth passes
       - Step 1: Call `checkBlocked()` — if blocked, return `{ status: 503, jsonBody: { error: "Service temporarily unavailable" } }`
       - Step 2: Read header `req.headers.get("x-app-password")`
       - Step 3: If header missing or doesn't match `process.env.APP_PASSWORD`:
         - Get `auth` collection, upsert document `{ _id: "lockout" }` incrementing `failureCount` by 1
         - If new `failureCount >= 30`, also set `blocked: true` (per D-11)
         - Return `{ status: 401, jsonBody: { error: "Unauthorized" } }`
       - Step 4: If password matches:
         - Upsert document `{ _id: "lockout" }` setting `failureCount: 0, blocked: false` (per D-12)
         - Return `null` (auth passed)

    4. **Update `api/local.settings.json`**:
       - Add `"APP_PASSWORD": "localdev123"` to Values
       - Add `"MONGODB_CONNECTION_STRING": "mongodb://localhost:27017"` to Values
       - Keep all existing values unchanged

    Import types: `import { HttpRequest, HttpResponseInit } from '@azure/functions'` and `import { MongoClient, Db } from 'mongodb'`
