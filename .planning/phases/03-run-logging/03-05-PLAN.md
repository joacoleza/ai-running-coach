    Flow:

       f. When status === 'done': show "Done — {runsExtracted} run(s) logged!", call `onComplete()` after 2 seconds
       g. When status === 'error': show error message from `errorMessage` field
    3. Cancel button visible at all stages (cancels polling, hides form)
    4. Progress states shown with appropriate icons/spinners
