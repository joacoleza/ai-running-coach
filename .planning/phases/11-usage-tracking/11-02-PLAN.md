<!-- GET /api/usage/me response shape (D-10) -->
```json
{
  "allTime":   { "cost": 2.47, "messages": 312 },
  "thisMonth": { "cost": 0.18, "messages": 23 },
  "monthly": [
    { "year": 2026, "month": 4, "cost": 0.18, "messages": 23 },
    { "year": 2026, "month": 3, "cost": 0.94, "messages": 118 }
  ]
}
```
Rows sorted newest-first (year DESC, month DESC).
If no usage events: { allTime: { cost: 0, messages: 0 }, thisMonth: { cost: 0, messages: 0 }, monthly: [] }

<!-- GET /api/users/usage-summary response shape (D-12) -->
```json
{
  "summary": {
    "<userId-string>": { "thisMonth": 0.18, "allTime": 2.47 },
    "<userId-string>": { "thisMonth": 0.00, "allTime": 0.00 }
  }
}
```
Users with no usage events are NOT included in summary (Admin.tsx shows $0.00 when key absent).

    const denied = await requireAuth(req);
    if (denied) return denied;
    const { userId } = getAuthContext(req);
    // ...
  };
}

<!-- Route registration pattern — match admin.ts exactly -->
app.http('usageMe', {
  methods: ['GET'],
  authLevel: 'anonymous',
  route: 'usage/me',
  handler: getUsageMeHandler(),
});

<!-- MongoDB aggregation for per-user breakdown -->
db.collection('usage_events').aggregate([
  { $match: { userId: new ObjectId(userId) } },
  { $group: {
      _id: { year: { $year: '$timestamp' }, month: { $month: '$timestamp' } },
      totalInputTokens: { $sum: '$inputTokens' },
      totalOutputTokens: { $sum: '$outputTokens' },
      totalCacheWriteTokens: { $sum: '$cacheWriteTokens' },
      totalCacheReadTokens: { $sum: '$cacheReadTokens' },
      messages: { $sum: 1 },
  }},
  { $sort: { '_id.year': -1, '_id.month': -1 } },
]).toArray()

Then compute cost per row using computeCost(model, ...) — use 'claude-sonnet-4-20250514' as model
since all current events use that model. Sum rows for allTime totals.
Determine thisMonth by filtering rows where _id.year === now.getFullYear() && _id.month === now.getMonth()+1.

<!-- Admin aggregation — all users -->
db.collection('usage_events').aggregate([
  { $group: {
      _id: {
        userId: '$userId',
        year: { $year: '$timestamp' },
        month: { $month: '$timestamp' },
      },
      totalInputTokens: { $sum: '$inputTokens' },
      totalOutputTokens: { $sum: '$outputTokens' },
      totalCacheWriteTokens: { $sum: '$cacheWriteTokens' },
      totalCacheReadTokens: { $sum: '$cacheReadTokens' },
  }},
]).toArray()

Then reduce into a map: userId.toString() → { thisMonth, allTime }.
