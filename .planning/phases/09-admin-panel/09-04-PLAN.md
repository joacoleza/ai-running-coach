```ts
function formatLastLogin(date?: Date | string): string {
  if (!date) return 'Never';
  const d = new Date(date);
  return d.toLocaleString('sv-SE', {         // 'sv-SE' produces ISO-like "YYYY-MM-DD HH:MM:SS"
    year: 'numeric', month: '2-digit', day: '2-digit',
    hour: '2-digit', minute: '2-digit', second: '2-digit',
    hour12: false,
  }).replace('T', ' ');
}
```

This gives output like `2026-04-19 14:35:22` — unambiguous and sortable. Apply this updated formatter in BOTH the desktop table cell AND the mobile card layout added in Task 1.
