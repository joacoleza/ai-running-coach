## Summary

Phase 03.1 achieves its goal. Two integration bugs fixed:

1. **COACH-03 (RunDetailModal stale closure)**: `handleAddFeedback` now uses `const responseText = await sendMessage(message)` directly for the insight, eliminating the stale-closure bug where it was reading from the `messages` array captured before the response arrived. The `messages` destructure was removed from `useChatContext()` to prevent future regression.

2. **COACH-04 (TrainingPlan raw XML)**: `handleGetFeedback` now strips self-closing XML tags via `.replace(/<[^>]+\/>/g, '').trim()` before saving `progressFeedback` to the API, preventing raw `<plan:update .../>` and `<app:navigate .../>` tags from appearing in the UI.

Both fixes are wired, tested, and verified. All 252 web tests pass. Build succeeds with no TypeScript errors.
